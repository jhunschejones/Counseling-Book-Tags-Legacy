Counseling Book Tags



Counseling Book Tags is all about you.



Why we're here:

Our goal with the counseling book tags application is to bring you a tool to make collecting and sharing counseling books easier. You can use it to add your own book tags or view tags and suggestions from others. We even have a comments section to include free-form details about how you've used a book in your practice.

Get Started:

To get started using the counseling book tags application, jump over to the home page to look through tags added by other users. A detailed description of the different application functions can be found below for those who prefer to read the instructions, or in case you have a question later about how a certain page works!

Application Layout:
  • Home page: The counseling book tags home page presents you with an alphabetized list of all the tags users have input for various books. You can choose to view all the tags and filter by your own criteria, or view a segment of the tags by letter then filter further using the filter-input box at the top of the page. Clicking directly on a tag will take you to the Tag results page.
  • Tag results page: When you click directly on a tag anywhere within the counseling book tags application, you will be taken to the tag results page for that specific tag. This page displays all the books that have been associated with your selected tag, as well as the other tags attached to those books. From this page you can either click directly on a book title to navigate to the book details page, or you can select additional tags to filter the results further. This is done by clicking on tags next to book titles on the page. All your selected tags will appear along the top of the page as you add and remove tags to filter the results.
  • Book details page: The book details page is available for any book in our tags database. This page allows you to see the cover of the book, as well as basic publication details, a description from Goodreads, and any tags or comments that have been added by other users. To add a comment or tag, click the "+" button next to tags or comments. From there you will be directed to the update page where you can log in with your google account to make modifications.
  • Update page: The update page allows you to add your own tags and comments to a book. We use Google authentication to make sure you are the only one who can delete tags or comments that you've added. We do not store any user-specific information in our databases, only a unique identifier supplied to us by Google. When you create a comment on the update page you will be prompted for a display name. This will be the name placed after your comment, and you are allowed to leave it blank if you prefer. Note: comments and tags will be displayed on the update page in white with an "X" next to them if you were the creator. This means you are able to delete these tags and comments if you so wish. Other users will see your tags and comments in teal, which indicates that they cannot be modified by other users.
  • Search page: The search page is where you look up a book to add your own tags. It will show books already tagged by other users, as well as books that have never been tagged yet. You can get to the search page by clicking the Search link in the navigation menu at the top of every page. You can search by title, author name or ISBN. Try to search by the most specific option you have available to make sure you find the book you are looking for. If the book exists in the database, it will appear in the first list. We encourage users to try to add tags to existing books rather than adding new books if there is a version of the book already in the database. This helps make tagged books easier to find by others later. If you need to add a new book, you can simply click the book title in the second list of books returned. You will receive a confirmation prompt if the book you clicked looks like it might be the same as one already in the database. If you want to use that book instead simply click it, otherwise you can choose to proceed with adding a new book. You will then be taken to the Update page where you can sign in with your Google credentials to add tags comments to the book.
Confidentiality:

Please note that all content posted in the application is public. As such, we do request that you use discretion when discussing particular client scenarios. Please remove all client names and personally identifiable information from any comments you leave so that others can benefit from your insight without having to worry about client confidentiality. If you find content on the application that you wish to be removed, please contact us immediately for assistance. If you include a page URL to the location you found the content, as well as a description of why you feel it should be removed, our moderators will investigate the issue get back to you.
